An Electron + React.js + Typescript app template. Find it at [`ilanolkies/electron-react-ts`](https://github.com/ilanolkies/electron-react-ts).

### Features

![](/img/electron-react-ts/front.png)

- Compiles [Typescript](https://www.typescriptlang.org/)
- Builds [React.js](https://reactjs.org/) with [`webpack`](https://webpack.js.org/)
- Runs on [Electron](https://www.electronjs.org/)
- Checks code syntax with [ESlint](https://eslint.org/) and [`standard-with-typescript`](https://github.com/standard/eslint-config-standard-with-typescript)
- Tests with [`ts-jest`](https://github.com/kulshekhar/ts-jest) and [Enzyme](https://enzymejs.github.io/enzyme/)
- Styles with [Sass](https://sass-lang.com/)

### How to use it

1. Browse to [`ilanolkies/electron-react-ts`](https://github.com/ilanolkies/electron-react-ts)
  ![](/img/electron-react-ts/repo.png)
2. Click on _use this teamplate_ on the top right corner
  ![](/img/electron-react-ts/use.png)
3. Update README badges urls
  ![](/img/electron-react-ts/shields.png)
4. [Install dependencies](https://github.com/ilanolkies/electron-react-ts#install-dependencies) and [run](https://github.com/ilanolkies/electron-react-ts#run) the app!
  ![](/img/electron-react-ts/dev.png)

Hope you like it!
